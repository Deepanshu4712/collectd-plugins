# Collectd Python Plugins

###### Download plugins to /opt/collectd/conf/

### Plugins Configs:

#### apache_perf
```xml
<Plugin python>
    ModulePath "/opt/collectd/plugins"
    LogTraces true
    Interactive false
    Import "apache_perf"

    <Module apache_perf>
        interval "300"
        port "<apache/httpd port number>"
        location "<location of stats e.g. server-status>"
        secure "False"
    </Module>
</Plugin>
```

#### apache_trans
```xml
<Plugin python>
    ModulePath "/opt/collectd/plugins"
    LogTraces true
    Interactive false
    Import "apache_trans"

    <Module apache_trans>
        accesslog "<path of access.log e.g. /var/log/apache2/access.log>"
        name "apache_trans"
        interval "100"
    </Module>
</Plugin>
```
